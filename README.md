# Веб-сервис "Путешествие дня"

## Короткое описание проекта
Приложение представляет из себя веб-сервис по покупке тура с помощью двух способов: оплате по дебетовой карте или выдаче кредита по данным банковской карты

## Начало работы
Для получения приложения необходимо
1. Сделать клон репозитория с https://github.com/netology-code/qa-diploma
Возможные варианты: через запуск команды git clone

## Prerequisites
Что нужно установить на ПК для использования 
1. Бразуер Chrome
1. Библиотеки Git с сайта github.com
1. Библиотеки Docker 
2. Библиотеки языка Java

## Установка и запуск
Пошаговый процесс установки и запуска
1. Вход в директорию, содержащую aqa-shop.jar
1. Запуск в терминале команды docker compose up
1. Запуск команды java -jar aqa-shop.jar для MySQL ИЛИ для PostgreSQL
1. Запуск симулятора в директории gate-simulator командой npm start
1. Вход в приложение - запуск в браузере команды localhost:8080
